À propos

Copilote RAG local pour la maintenance industrielle — répond à des questions sur les données NASA CMAPSS (capteurs, durée de vie) et les catalogues Schaeffler / SKF (4 343 pages).

✓ Il répond à : questions factuelles (« combien de moteurs dans FD001 ? »), tendances de capteurs, calculs sur les données structurées, questions sur les roulements / lubrification / montage.

✗ Il ne répond PAS à : questions hors domaine (météo, cuisine, etc.) — il le dit explicitement.
